✈️ Viagem APP — UI/UX Design
📌 Sobre o Projeto

O Viagem APP é um protótipo de aplicativo mobile desenvolvido no Figma com o objetivo de facilitar o planejamento de viagens, oferecendo uma experiência intuitiva e visualmente agradável para os usuários.

O projeto foi criado com foco em usabilidade, organização de informações e navegação simples, permitindo que o usuário encontre destinos, explore opções e planeje sua viagem de forma prática.

🎯 Objetivo

Desenvolver uma interface moderna e funcional para um aplicativo de viagens, proporcionando:

Navegação intuitiva
Experiência agradável ao usuário
Organização clara das informações
Facilidade na busca e escolha de destinos
👤 Público-alvo
Pessoas que gostam de viajar
Usuários que buscam praticidade no planejamento
Turistas iniciantes ou frequentes
🧠 Conceito de UX/UI

O projeto foi pensado com base em princípios de UX (Experiência do Usuário) e UI (Interface do Usuário), garantindo:

Layout limpo e minimalista
Hierarquia visual clara
Componentes consistentes
Fluxo de navegação simples

A interface prioriza uma experiência fluida, evitando sobrecarga de informações e facilitando a tomada de decisão do usuário .

🔄 Fluxo do Usuário

O fluxo principal do aplicativo inclui:

Acesso inicial
Exploração de destinos
Visualização de detalhes
Seleção de opções
Planejamento da viagem

O fluxo foi estruturado para ser direto e intuitivo, reduzindo fricções durante o uso.

🎨 Design e Interface
Estilo moderno e minimalista
Uso de cores suaves para melhor experiência visual
Tipografia legível
Componentes reutilizáveis
Organização em grids e espaçamentos consistentes
